# 🍔 Online Food Delivery System

A backend application for an online food delivery platform built using Spring Boot.

The system manages users, restaurants, orders, and delivery services through REST APIs in a modular structure.

## 🚀 Tech Stack
- Java
- Spring Boot
- Maven
- REST APIs

## 📌 Features
- User management
- Restaurant management
- Order processing
- Modular service structure

This project demonstrates backend development using Spring Boot and RESTful architecture.
